Sistem Pempek N'Kitchen

Sistem Pemesanan Makanan Berbasis Web

Latar Belakang & Tujuan

Latar Belakang: N-Kitchen menghadapi kendala dalam pencatatan pesanan manual yang rawan kesalahan dan promosi yang terbatas.

Tujuan: Membangun sistem pemesanan berbasis web yang terintegrasi dengan Payment Gateway (Midtrans) dan API Logistik (Biteship) untuk efisiensi operasional dan perluasan pasar.

Metodologi

  • Pengembangan: Waterfall Model.
  • Backend: Laravel 11.
  • Frontend: Blade Template, Tailwind CSS.
  • Database: MySQL.
  • Integrasi: Midtrans (Pembayaran), Biteship (Ongkir).
